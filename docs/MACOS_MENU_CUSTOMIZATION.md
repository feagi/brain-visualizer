# macOS Native Menu Bar Customization Guide

## ⚠️ Current Status

**Menu customization is currently disabled** because the `DisplayServer` global menu API constants (`SYSTEM_MENU_APP`, `FEATURE_GLOBAL_MENU`) do not exist in Godot 4.5 stable.

The default macOS menu bar is automatically generated by Godot and includes:
- **"About Brain Visualizer"** (system default)
- **"Preferences..."** (system default, but doesn't connect to our settings)
- **"Quit Brain Visualizer"** (Cmd+Q, handled by system)

## Overview

Godot's `DisplayServer` API should allow customization of the macOS native menu bar, but the exact API varies by Godot version. This guide documents the approach once the correct API is identified.

## Finding the Correct API

To find the correct API for your Godot version:

1. **Check Godot Documentation**:
   - [Godot 4.5 DisplayServer Class Reference](https://docs.godotengine.org/en/4.5/classes/class_displayserver.html)
   - Search for "global_menu" or "menu" methods

2. **Check Available Methods**:
   ```gdscript
   # Print all DisplayServer methods
   var methods = DisplayServer.get_method_list()
   for method in methods:
       if "menu" in method.name.to_lower():
           print(method.name)
   ```

3. **Check Available Constants**:
   ```gdscript
   # Print all DisplayServer constants
   var constants = DisplayServer.get_property_list()
   for prop in constants:
       if "MENU" in prop.name:
           print(prop.name)
   ```

4. **Alternative Approaches**:
   - Use `OS.alert()` for simple dialogs (works, but not menu items)
   - Create custom in-app menu UI instead of native menu bar
   - Wait for Godot 4.6+ which may have improved menu API

## How It Works (When API is Available)

### 1. Menu Setup (`_setup_native_menu()`)

Called deferred in `_ready()`:
```gdscript
if OS.get_name() == "macOS":
    call_deferred("_setup_native_menu")
```

### 2. Getting the Native Menu

```gdscript
# Check if global menus are supported
if not DisplayServer.has_feature(DisplayServer.FEATURE_GLOBAL_MENU):
    return

# Get the app menu (returns menu ID, or -1 on error)
var native_menu = DisplayServer.global_menu_get_system_menu(DisplayServer.SYSTEM_MENU_APP)
if native_menu == -1:
    push_warning("Failed to get native menu")
    return
```

### 3. Adding Menu Items

```gdscript
# Basic menu item (no shortcut)
var callback = Callable(self, "_on_menu_handler")
DisplayServer.global_menu_add_item(native_menu, "Menu Label", callback)

# Menu item with keyboard shortcut (Cmd+Shift+S)
var shortcut = KEY_MASK_META | KEY_MASK_SHIFT | KEY_S
DisplayServer.global_menu_add_item(
    native_menu,
    "Save As...",
    Callable(self, "_on_save_as"),
    Callable(),  # key_callback (optional)
    "",          # tag (optional)
    shortcut     # accelerator
)

# Separator
DisplayServer.global_menu_add_separator(native_menu)
```

### 4. Menu Item Handlers

```gdscript
func _on_menu_handler() -> void:
    print("Menu item clicked!")
    # Your code here
```

## API Reference

### `DisplayServer.global_menu_add_item()`

**Signature:**
```gdscript
global_menu_add_item(
    menu_root: int,
    label: String,
    callback: Callable,
    key_callback: Callable = Callable(),
    tag: Variant = "",
    accelerator: Key = 0,
    index: int = -1
) -> int
```

**Parameters:**
- `menu_root`: Menu ID from `global_menu_get_system_menu()`
- `label`: Text displayed in menu
- `callback`: Function to call when clicked (`Callable(self, "function_name")`)
- `key_callback`: Optional callback for key events
- `tag`: Optional metadata string
- `accelerator`: Keyboard shortcut (e.g., `KEY_MASK_META | KEY_COMMA` for Cmd+,)
- `index`: Position to insert (-1 = append to end)

**Returns:** Item ID (or -1 on error)

### Keyboard Modifiers

- `KEY_MASK_META` - Command (⌘)
- `KEY_MASK_SHIFT` - Shift (⇧)
- `KEY_MASK_CTRL` - Control (⌃)
- `KEY_MASK_ALT` - Option (⌥)

### Common Keyboard Shortcuts

```gdscript
# Cmd+Q (Quit)
KEY_MASK_META | KEY_Q

# Cmd+, (Preferences)
KEY_MASK_META | KEY_COMMA

# Cmd+S (Save)
KEY_MASK_META | KEY_S

# Cmd+Shift+S (Save As)
KEY_MASK_META | KEY_MASK_SHIFT | KEY_S

# Cmd+N (New)
KEY_MASK_META | KEY_N

# Cmd+O (Open)
KEY_MASK_META | KEY_O
```

## Examples

### Add a "Help" Menu Item

```gdscript
func _setup_native_menu() -> void:
    # ... existing code ...
    
    # Add Help menu item
    DisplayServer.global_menu_add_item(
        native_menu,
        "Help",
        Callable(self, "_on_menu_help")
    )

func _on_menu_help() -> void:
    OS.shell_open("https://feagi.org/docs")
```

### Add a Submenu

```gdscript
# Create submenu
var submenu_id = DisplayServer.global_menu_add_submenu_item(
    native_menu,
    "View",
    "view_submenu"
)

# Add items to submenu
DisplayServer.global_menu_add_item(
    submenu_id,
    "Show FPS",
    Callable(self, "_toggle_fps")
)
```

### Add Menu Item with Checkmark

```gdscript
var show_fps_id = DisplayServer.global_menu_add_item(
    native_menu,
    "Show FPS",
    Callable(self, "_toggle_fps")
)

# Later, toggle checkmark
DisplayServer.global_menu_set_item_checked(native_menu, show_fps_id, true)
```

### Disable/Enable Menu Items

```gdscript
# Disable menu item
DisplayServer.global_menu_set_item_disabled(native_menu, item_id, true)

# Enable menu item
DisplayServer.global_menu_set_item_disabled(native_menu, item_id, false)
```

## Best Practices

1. **Always check platform**: Only customize on macOS
   ```gdscript
   if OS.get_name() != "macOS":
       return
   ```

2. **Use deferred calls**: Don't block initialization
   ```gdscript
   call_deferred("_setup_native_menu")
   ```

3. **Check for errors**: Verify menu operations succeed
   ```gdscript
   var menu_id = DisplayServer.global_menu_get_system_menu(...)
   if menu_id == -1:
       push_warning("Menu setup failed")
       return
   ```

4. **Follow macOS conventions**:
   - "Preferences..." (not "Settings")
   - Cmd+, for Preferences
   - Cmd+Q for Quit (handled by system)
   - Use separators to group related items

5. **Handle missing UI managers**: Check if dependencies exist
   ```gdscript
   if _UI_manager and _UI_manager.window_manager:
       _UI_manager.window_manager.spawn_options_menu()
   ```

## Troubleshooting

### Menu items don't appear
- Check console for warnings
- Verify `DisplayServer.has_feature(DisplayServer.FEATURE_GLOBAL_MENU)` returns true
- Ensure menu setup runs after app initialization (use `call_deferred`)

### Shortcuts don't work
- Verify modifier keys are correct (`KEY_MASK_META` for Cmd)
- Check for conflicts with system shortcuts
- Ensure accelerator parameter is set correctly

### Callbacks don't fire
- Verify `Callable(self, "function_name")` syntax
- Check function signature matches (no parameters for menu callbacks)
- Ensure function is defined in the same script
